Introduction

In most cases you can change all your settings by using the HS Prims menu.


To use commands you type them in chat while standing next to the HS Prim and hit Enter.

To add information such as names or set up networking you will need to know these commands.

All the commands bellow are assuming that your settings card has the default setting for channel. That is channel 8.
If you wish to change the channel the Prim accepts commands on or sends normal messages see the "Channel" notecard inside it.

You must be within 20 meters when typing commands for the HS Prim to hear you. If you shout the command you must be within 100m. It may not work over region borders.

Green entries are not on the menus, many are advanced commands.

Basic Commands


/8menu for the popup control menu (shout this if you aren't near the Prim)
/8reset force the Prim to read and process the Settings notecard (It doesn't by default anymore) without loosing current names or old network HSIDs in memory.
/8status summary of status for all scripts
/8off
 Turn Off
/8public Turn on for public areas like shops and clubs. Attacks only Targets.
/8guests Turn on for private areas like homes. Attacks all not in Guests or Friends list or not Group Safe member.
/8friends Turn on for private areas like homes for Friends ONLY. Attacks ANYONE not in Friends list. Will attack Group Safe members and Guests.
/8hsid gives the HS Prims's HSID for networking. Shout this and you will know which HS Prims are within 100m. Will also give the Prims distance.

Adding/Removing Names and Network


/8addt full name
/8addt user.name
/8addt ★♥ Display Name ♥★
Adds a name to Target List using either legacy Full Name or new user.name
Do not use Display Names.
/8addt *companyname Discontinued in Version 12.
/8addg full name
/8addg user.name
/8addg ★♥ Display Name ♥★
Adds a name to the Guest List using either legacy Full Name or user.name
Guests are allowed when online in 'Guests Mode', and not allowed when online in 'Friends Mode'. You can also use groups to control who is allowed to visit when in Guest Mode. See Groups.
Do not use Display Names.
/8addf full name
/8addf user.name
/8addg ★♥ Display Name ♥★
Adds a name to the Friends List using eithe legacy Full Name or user.name
Friends are always allowed to visit. Note: Group Safe members and Guests are NOT allowed to visit when online in Friends Mode. Do not use Display Names.
/8addo full name
/8addo user.name
/8addo ★♥ Display Name ♥★
Adds a name to the Operator List using eithe legacy Full Name or user.name
You can also use groups to control who is an Operator. See Groups.
Do not use Display Names.
/8addn HSID Adds a HSID to the Network List. See Networking.
/8delt name Deletes name from Target List. Click Names on the menu for better method.
/8delg name Deletes name from Guest List. Click Names on the menu for better method.
/8delf name Deletes name from Friends List. Click Names on the menu for better method.
/8delo name Deletes name from Operator List. Click Names on the menu for better method.
/8del nameDeletes name from Friend, Guest & Operator List with one command.
/8deln HSID Deletes HSID from Network List. Click Names on the menu for better method.
/8deltv Deletes ALL Targets and ALL Visitors (Guests and Friends) INCLUDING YOU. Re-add yourself just incase.
/8delop Deletes ALL Operators, then adds you as an Operator.
/8delnr Deletes ALL Network HSIDs.
/8sendtv
 Send all Targets and Visitors (Guests and Friends) names to all Prims in this network.
/8sendo
 Send all Operators names to all Prims in this network. v12.018+
/8lb h.h avatar
 Land Ban avatar for h.h hours. To unban you will need to remove the name using a SL version 2+ client in About Land / Access. These names are not stored in the HS Prim. It simply passes the request onto SL for you. Requires v12.015+
avatar can be a User Name or Key.
h.h is between 0.0 and 144.0 hours, 0 means forever.
/8tp avatar
 Teleport Home avatar . These names are not stored in the HS Prim. It simply passes the request onto SL for you. Requires v12.017+
avatar can be a User Name or Key.

Changing Settings

Most settings are changed on the menu under Settings. These commands below are not on the menu.

/8visible
/8v 		Toggle on/off both the visibility of the HS Prim and its floating text.
/8up=number How far above the Prim to limit attacks (for apartments & multilevel buildings, skyboxes) in meters.
/8down=number How far below the Prim to limit attacks (for apartments & multilevel buildings, skyboxes) in meters.
/8amsg Attacked message Sets attacked intruders popup to say "Attacked message".
/8wmsg Warning message Sets intruders warning popup to say "Warning message".
/8dmsg Fake Display
              Name message 		When a detected avatar, who is not an Operator, has a Display Name exactly matching an Operator they will be optionally Teleported Home. Then a popup will say "Fake Display Name message". See SETTINGS / Scanner / DispProt
/8gwmsg Guests message Overrides "Warning message" with "Guests message" when guest/group members try to enter while HS Prim is in Friends only Mode. If not set then "Warning message" from /8wmsg is used.
/8warn on|off enable/disables the warning of intruders before attacking.
/8oi on|off
 enables/disables HS Guardian HUD communication with the HS Prim.
Leave off unless needed.
/8ban on|off
 enables/disables banning a person from the land for 30 minutes. Script land banned people will see blinking "NO ENTRY" ban lines but may not be listed in About Land/Access unless you use a modern Viewer 2 client. The default 0.5 hours can be changed with /8bantime=n.n
/8bantime=number
 Ban time in hours when Ban is set on. 0.0 hours means permanently. SL has a maximum limit of 144.0 hours except when permanent is used.
Disable with /8ban off and enable with /8ban on
/8web on|off
"New Web URL" and "Web page changed" Instant Messages sent to last Operator on or off.
/8nthop on|off
When Notify Target pop-up is in use (see /8nt below) should the pop-up hide the Operator's name who activated the Prim. (Version 13.031+)
/8slw scriptmb scriptcount
                scripttime scripttimecount
 Script Limit Warnings for all avatars. Exceeding these limits results in them seeing a pop-up asking them to detach some scripts.

If less than 4 parameters are supplied the script assumes the missing paramaters at the end are unchanged. Any parameter that is a * (asterix) is unchanged. Any parameter that is -1 disables that parameter.

scriptmb is the amount of attached script memory in megabytes before seeing a warning.
scriptcount is the count of attached scripts before seeing a warning.
scripttime is the simulator CPU time in µs (micro-seconds) per simulator frame of the avatar's scripts before they see a warning. The simulator under ideal conditions produces 45 frames a second, therefore 22,222µs per frame. A portion of this 22,222µs is devoted to running scripts. An avatar who has a script time of 1,000µs would be using 4.5% of the simulators processing time.
scripttimecount is the number of consecutive scans the scripttime has been crossed before action is taken.


Examples:
/8slw -1 -1 -1 -1
 disables all script warnings.
/8slw 0
 shows warning popup to scripted avatars (because any script would be more than 0MB).
/8slw 3 10 warns scripted avatars who have more than 3MB of scripts OR more than 10 scripts.
/8slw 3 would enable script memory check and warn at over 3MB, ignore the script count and CPU time check.
/8slw -1 10 would enable script count check and warn at over 10 scripts and ignore the script memory and CPU time check.
/8slw -1 -1 1000 would enable script CPU time checks and warn at over 1000µs usage and ignore the script memory and count check.
/8slw 10 -1 1000 10
 would enable script memory check and warn at over 10MB, ignore the script count check, and check for script usage over over 1000µs usage for 10 consecutive scans.
/8slt scriptmb scriptcount
               scripttime scripttimecount
 Script Limit Teleports for non-Friend avatars. Exceeding these limits results in them being teleported home.

This works in the same fashion as /8slw above except the result will be a teleport home. See /8slw above for examples.
/8stmon
 Enable monitoring of script time usage of avatars within range that are over the scripttime warn or TP limits. This is whispered so you need to be within 10m of the HS Prim to see the text.
/8age daysold
/8age daysold hours
/8age off
Attack any avatar equal to or under daysold old for the next hours.
Ignoring hours will default to until the year 2038.
/8age off disables. Version 12.020+
 

Menu Buttons are Commands

Most menu buttons are commands also but it is probably easier to use the menus.
Here are some examples:

/8sleep Enables/Disables sleeping when Operator is within range. Useful for some Estates where security systems must be off when no one is there.
/8st=number Set the Scan Timer rate in seconds. Lowest should be 6, Lindens
recommend 10 seconds
/8ar=number Limit Armed Range (ie. when in Public or Guests or Friends Mode). Non-zero number.
/8thisland
 Limit Armed Range to this land parcel only.
/8landowners
 Limit Armed Range to all of this land owner's parcels within this region.
/8region
 Limit Armed Range to the whole region.
The Prim will still repect Foreign Land Safe, Other Land Safe, Up and Down restrictions.
/8sr=number
 Limit NoAttack mode range. Non-zero number.
/8teleport make the weapon be Teleport Home
/8eject make the weapon be Eject
/8gs on|off Group Safe on or off
/8fls on|off Foreign Land Safe on or off. Foreign Land is land with a different owner to the land under the HS Prim.
/8ols on|off Other Land Safe on or off. Other land is any other land, even if it has the same name.
/8nop im|menu|off Notify Operator who turned it on (IM = Instant Message, Menu = Popup Blue menu (menu only works after you visit the Region the Prim is in), Off.
/8nt on|off Notify Target on or off
/8log See the log menu
/8vlog Visitors Log
/8clog Commands Log

Note: A few more examples are /8weapon, /8listnear, /8groupsafe, /8public, /8names, /8visitors etc etc
