Avatar Authority

For version 12+ a new letter has been added to indicate which name type was a match.
uF/ Friend based on User Name match. Friends are allowed any time.
fF/
Friend based on legacy Full Name match. Friends are allowed any time.
kO/ Owner of the HS Prim based on avatar Key.
kG/ Same Group as the HS Prim and Group Safe is on. On the HS Prim menu see Settings / Weapon
ug/ Guest based on User Name match. Guests are allowed when online in 'Guest Mode'.
fg/ Guest based on legacy Full Name match. Guests are allowed when online in 'Guest Mode'.
--/ Avatar not in a List, neither Friend, Guest or Target.
uT/ Targeted based on User Name match.
fT/ Targeted based on legacy Full Name match.
dT/ Targeted based on Display Name match.
Note: These letter symbols will show in the Visitor Log also.

???㎆/ Megabytes of memory usage. Rounded to the nearest Megabyte. 1MB = 1,000,000bytes. (Version 12.016+)
/??? Number of scripts (running and stopped) attached to the avatar. (Version 12.016-13.033)
/???µs Micro-seconds of simulator CPU time used by the avatars scripts every simulator frame. The simulator under ideal conditions generates 45 frames a second. Therefore each frame takes 22.222ms (milli-seconds) or 22,222µs (micro-seconds). An avatar using thousands of µs may be contributing to region lag. (Version 13.035+)

R
 Range is set to whole Region.
LO
 Range is set to this Land Owner's parcels anywhere in this region.
TL
 Range is set to This Land parcel only.
